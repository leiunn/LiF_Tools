# Introduction
LiF Tools is a small program that helps you manage your characters for the games Life is Feudal: Your Own and Life is Feudal: MMO.

# Main Features
- Add all your existing characters with their attributes and skills
- Plan your characters
- Manage and plan skills
- Manage your herbal specifications
- Create reports of your herbal specifications
- Save and load your created data
...

# Installation
1. Download the .msi file and run it
2. Choose the install directory and install the application
3. Run the application from your start menu, desktop or installation directory
